A powerful and user-friendly **Personal Budget Tracker** built with Python. Take control of your finances with beautiful visualizations, insightful summaries, and easy transaction management!

## ✨ Features

### 📊 **Financial Management**
- ✅ Add income and expense transactions
- ✅ Categorize transactions (Food, Rent, Salary, etc.)
- ✅ View complete financial summary
- ✅ Track balance and savings rate

### 📈 **Beautiful Visualizations**
- ✅ Expense breakdown pie charts
- ✅ Income vs Expenses bar charts
- ✅ Monthly spending trends
- ✅ Interactive matplotlib graphs

### 🔍 **Advanced Features**
- ✅ Filter transactions by date range
- ✅ Delete/Edit transactions
- ✅ Colorful terminal interface
- ✅ CSV data storage
- ✅ Financial health insights
